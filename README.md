# :bar_chart: Ecomerce Data Analysis

## O projekcie
Projekt skupia się na analizie danych platformy e-commerce.

## Stack Technologiczny 

* **System operacyjny** Windows Subsystem for Linux (**WSL 2/ Ubuntu**)
* **Baza danych** PostgreSQL
* **Języki** SQL, Python (Pandas, SQLAlchemy, Matplotlib, Seaborn)
* **Narzędzia** Jupyter Notebook, Git
